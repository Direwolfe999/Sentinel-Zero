API ENDPOINT SPECIFICATIONS
================================================================================

ENDPOINT: POST /api/analyze

REQUEST SCHEMA:
  {
    "mode": "finance|life|business"          (required)
    "text": "string (1-10000 chars)"          (required)
    "image": "data:image/...;base64,..."     (optional)
    "audio": "data:audio/...;base64,..."     (optional)
    "video": "data:video/...;base64,..."     (optional)
  }

RESPONSE (SUCCESS):
  {
    "success": true,
    "data": {
      "risk": "HIGH|MEDIUM|LOW",
      "insight": "string",
      "recommendation": "string",
      "confidence": 0.0-1.0
    },
    "timestamp": "ISO8601"
  }

RESPONSE (ERROR):
  {
    "success": false,
    "error": "error message",
    "timestamp": "ISO8601"
  }

STATUS CODES:
  - 200 OK: Successful analysis
  - 400 Bad Request: Invalid input
  - 500 Internal Server Error: Processing error
